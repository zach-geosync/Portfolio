# Zachary Brown Geospatial Portfolio

My geospatial portfolio: https://zach-geosync.github.io/Portfolio/

Welcome to my geospatial portfolio, built with HTML, CSS, and GitHub Pages. This site highlights selected projects in remote sensing, GIS, wetland science, UAV imagery, environmental field data, and spatial analysis using lightweight web-ready images and a résumé PDF.

## Featured Work

This portfolio includes projects focused on:

- Wetland methane flux variability using satellite indices and eddy covariance observations
- NASA DEVELOP forest dynamics and oak-hickory restoration suitability mapping
- UAV-based corn remote sensing and vegetation index analysis
- Post-Hurricane Helene landslide detection using Google Earth Engine
- Urban Heat Island modeling with Geographically Weighted Regression
- Bald cypress knee density analysis using field surveys and LiDAR terrain data
- Stem-mediated methane flux modeling from bald cypress trees
- Murphy’s Pond wetland assessment and ecological interpretation

## Tools and Methods

Core tools and methods represented across the portfolio include:
